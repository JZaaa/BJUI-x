第三方插件：Bootstrat-select —— 美化下拉选择框

本节主要介绍表下拉选择框插件 - Bootstrat-select，插件地址：http://silviomoreto.github.io/bootstrap-select/。

初始化

  • Data属性：select添加属性data-toggle="selectpicker"后即可触发。

    DOM示例：　

    示例代码：

                        <select name="doc-select" data-toggle="selectpicker">
                        <option value="A">选项A</option>
                        <option value="B">选项B</option>
                        <option value="C">选项C</option>
                    </select>
  • jQuery API：http://silviomoreto.github.io/bootstrap-select/
